# Minimal Home
## Work efficient. Be minimal.

### A minimalist replacement for chrome's new tab.

# How to install:
1. Download the zip archive
2. Go to your extensions page (chrome://extensions)
3. Turn on developer mode
4. Click "Load unpacked"
5. Choose the folder with this extension's files
6. Go to a new tab and click Customize Chrome
7. Turn off "Show footer on New Tab page"
